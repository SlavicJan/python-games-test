# Way of the Warrior — Pygame starter (isometric + Diablo-like UI)

Это **MVP-стартер**: изометрическая сетка, герой, клики по клеткам (A* путь), камера, UI-рамка/оверлей, миникарта, инвентарь/диалог, портал.

## Быстрый запуск (Windows)

1) **Дважды клик** по `start.bat` (первый запуск поставит зависимости/venv).
2) Потом просто запускай `run_game.bat`.

### Если не хочется venv
Можно запустить прямо так (из папки проекта):

```bat
python main.py
```

Но тогда pygame должен быть установлен в твоём глобальном Python.

## Управление

- **ЛКМ** — идти в клетку (строит путь и идёт по точкам)
- **WASD / стрелки** — камера
- **C** — центрировать камеру на герое
- **I** — инвентарь (показ/скрыть)
- **O** — диалог/квестовое окно (показ/скрыть)
- **F1** — debug (сеткa, координаты, safe-rect)
- **E** — активировать портал (если стоишь рядом)
- **T** — телепорт к порталу (быстрый тест)
- **ESC** — выход

## Что уже есть
