# Way of the Warrior — starter (Pygame) + AI UI pack

Это та же заготовка, но я добавил папку `assets/` и в коде включил отрисовку UI поверх карты.
UI нарезан из твоих двух сгенерированных картинок (концепты).

## Запуск
pip install -r requirements.txt
python main.py

## Управление
- ЛКМ или ПКМ по земле: герой идёт к точке (A*)
- WASD/стрелки: сдвиг камеры
- I: инвентарь (показывает наложение панели справа)
- O: диалог/меню (показывает окошко сверху)
- ESC: выход

## Как дальше делать графику, если ты сам не художник
1) Генеришь концепт (как ты сделал).
2) Я нарезаю/чистю под ассеты (PNG) и кладу в `assets/`.
3) Мы по одному экрану заменяем на реальные UI-элементы:
   - рамка (overlay)
   - панели (инвентарь/персонаж)
   - иконки/слоты
   - тайлы пола/стен (тайлсет)
4) Когда механика готова — можно улучшать арт без переписывания игры.

### Какие файлы менять
- assets/ui_frame_overlay.png — рамка поверх всего
- assets/ui_inventory_panel.png — панель инвентаря (пока просто картинка)
- assets/ui_dialog.png — окно диалога/выбора


## Asset sizes (important)

- Best results if UI assets are generated in **16:9** (e.g. 1280×720 or 1920×1080).
- If you drop in larger images, the code now auto-fits them, but mismatched aspect ratios may crop the edges.


## v3 изменения
- UI выравнивается относительно 'окна' рамки (safe rect)
- Клики вне окна игнорируются
- Портал видно (в центре карты). Подойди и нажми E — спавнит врагов (пока кружочки)
