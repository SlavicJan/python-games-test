# Way of the Warrior — quick start (Windows)

## 1) Install deps (once)
Double-click **install_deps.bat**.

## 2) Run game
Double-click **run_game.bat**.

If PowerShell blocks scripts (ExecutionPolicy) — no worries: the .bat files do **not** need PowerShell activation.

## Controls
- **LMB / RMB**: set destination tile (click on the grid)
- **WASD / arrows**: move camera
- **C**: center camera on hero
- **T**: teleport hero to portal
- **E**: spawn enemies (only when you are near the portal)
- **I**: inventory toggle
- **O**: dialog window toggle
- **F1**: debug overlay
- **ESC**: quit
